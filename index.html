Aufgabe

Erstellen sie für diese Webseite ein CSS Layout.
Das CSS File kann über die URL eingebunden werden.

Ob sie ein 2 oder 3 Spalten Layout wählen bleibt ihnen überlassen.
Uns geht es hierbei nicht um die grafische Gestaltung als vielmehr um die technische Umsetzung.

Auf jedenfall sollte die Navigation unter Benutzung der :hover Pseudoklasse gestyled werden, der aktive Menüpunkt ist über die Bodyklasse gesetzt.
Weiters sollte zumindestens im #header die Überschrift mittels "imagereplacement" durch ein Bild ersetzt werden.

Achten sie auch bitte darauf das der Text gut zu lesen ist.

Targetbrowser:
Firefox
Internet Explorer 6
Internet Explorer 7
Internet Explorer 8

Unter Links finden sie Internet Explorer 6 als download (standalone) sowie einige hilfreiche Links.
